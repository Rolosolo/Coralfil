# Coralfil Project Guide

## 🛠️ Build & Development
- **Dev**: `npm run dev`
- **Build**: `npm run build`
- **Lint**: `npm run lint`
- **Test**: `npm run test`

## 🎯 Global Constraints
- **Branding**: Always use **Coralfil** (one 'L' at the end).
- **Domain**: Canonical domain is **coralfil.com**.
- **Market Focus**: Specialize in **BC Shellfish Growers** and sessile marine species resilience.
- **Styling**: Vanilla CSS (no Tailwind unless requested). Modern, premium, glassmorphism aesthetics.
- **Components**: 
  - Use server components for static content (see `/research`).
  - Use `motion-client.tsx` for framer-motion animations.

## 🧠 Context Management
- This project uses **Strategic Compacting**. 
- Before compacting, update `src/PROJECT_MEMORY.md` with recent architectural decisions or research findings.
